# AddNewToolbox

The "AddNewToolbox" component is a replica of the toolbox used in SharePoint for Modern Sites, e.g. to place a new web part. 

## Arguments

<ArgTypes of={AddNewToolbox} />

## Usage

The React Autocomplete Component is useful in scenarios where users need to input text efficiently by selecting from a predefined list of options. It enhances user experience by reducing typing efforts and ensuring accurate inputs.

### Import

```typescript
import { AddNewToolbox } from '@spfxappdev/fluentui-react-controls';
```

### Sample Code

```typescript
import { AddNewToolbox } from '@spfxappdev/fluentui-react-controls';

<AddNewToolbox
    items={[
        { key: "addKey", title: "Example 1", icon: "Add" },
        { key: "editKey", title: "Example 2", icon: "Edit" },
        { key: "deleteKey", title: "Example 3", icon: "Delete" }
    ]}
/>
```
