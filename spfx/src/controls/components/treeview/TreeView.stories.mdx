# TreeView

The `TreeView` component is a powerful accessible user interface element that displays hierarchical data in a tree-like structure. 
It provides users with an intuitive way to navigate through nested categories, folders or other hierarchical data sets. 
It offers extensive customization options and is fully keyboard accessible, allowing users to seamlessly interact with it using only their keyboard.



## Arguments

<ArgTypes of={TreeView} />

## Usage

### Import

```typescript
import { TreeView } from '@spfxappdev/fluentui-react-controls';
```

export const BasicTemplate = (args) => (
  <div>
    {initializeIcons()}
    <TreeView {...args} />
  </div>
);
