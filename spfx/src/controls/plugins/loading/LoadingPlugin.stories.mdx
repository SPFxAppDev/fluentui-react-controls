# Loading Plugin

The `Laoding Plugin` is a utility that simplifies the process of opening a loading screen within React applications without the need for complex state management. 
With just a single line of code, developers can use this plugin to open loading screens.

{/* ## Arguments

<ArgTypes of={Loading} /> */}
## Usage 

### Sample Code

```typescript
import { Plugins } from '@spfxappdev/fluentui-react-controls';

<PrimaryButton
  onClick={() => {
      Plugins.Loading.show();

      var timer = setTimeout(() => {
          Plugins.Loading.hide();
          timer = void 0
      }, 3000);
  }}
  >
  Show Loading
</PrimaryButton>
```
