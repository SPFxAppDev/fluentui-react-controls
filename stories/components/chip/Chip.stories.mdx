# Chip

The `Chip` component is a small, interactive element used to represent an input, attribute, or action. It can include icons and be customized in various ways.

## Arguments

<ArgTypes of={Chip} />

### Import

```typescript
import { Grid } from "@spfxappdev/fluentui-react-controls";
```

## Examples

### Basic Chip

<Canvas>
  <Story
    name="Basic Chip"
    args={{
      label: "Basic Chip",
    }}

  </Story>

### Chip with Icons

<Canvas>
  <Story
    name="Chip with Icons"
    args={{
      label: "Chip with Icons",
      prefixIcon: "Emoji2",
      suffixIcon: "Cancel",
    }}
  >
    {(args) => (
      <Chip
        onPrefixClick={() => alert("Prefix icon clicked")}
        onSuffixClick={() => alert("Suffix icon clicked")}
        onClick={() => alert("Chip clicked")}

      />
    )}
  </Story>
</Canvas>
